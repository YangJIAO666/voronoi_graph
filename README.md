## Main modules

- `graph_builder.py` — build the weighted graph from store and socioeconomic data.
- `voronoi_detector.py` — core Voronoi-based community detection implementation and community visualisation.
- `modularity.py` — Newman modularity and related utilities.
- `optimization_multiobjective.py` — multi-objective optimisation for edge weights and the scale parameter.
- `multiscale_analysis.py` — multi-scale community-structure analysis.
- `sensitivity_analysis.py` — sensitivity analysis with basic plots and printed summaries.
- `investment_analysis.py` — investment analysis based on detected communities and related visualisation.
- `evaluation_fixed_communities.py` — evaluation of different methods at fixed numbers of communities.
- `timing_benchmark.py` — timing benchmarks for selected methods.
